## About the Course

This repository documents the practical journey through **Field Work (Course Code: 4708)**, a hands-on course centered on **Advanced Generative AI** — covering the tools, protocols, and frameworks that power modern agentic AI systems.

The course bridges theory and practice, guiding learners from foundational API concepts to building production-style AI agents using the latest orchestration frameworks in the GenAI ecosystem.

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 1 | **API** | Fundamentals of building and consuming APIs for AI-powered applications |
| 2 | **LangChain** | Building LLM-powered applications with chains, tools, and agents |
| 3 | **LangGraph** | Designing stateful, graph-based multi-step agent workflows |
| 4 | **MCP (Model Context Protocol)** | Connecting AI models to external tools, data, and services |
| 5 | **And Others** | Additional emerging concepts and tools in the GenAI landscape |
